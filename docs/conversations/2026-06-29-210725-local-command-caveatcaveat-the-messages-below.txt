  added.

  You run these once (need your CF account), then paste the IDs into
  wrangler.jsonc:
  cd server
  wrangler d1 create frimmy          # → database_id
  wrangler kv namespace create DIFFS # → id
  wrangler d1 migrations apply frimmy --local   # and --remote for prod
  bun run cf-typegen                 # regen binding types

  Auth = Cloudflare Access (not in-Worker OAuth): create a self-hosted Access
  app over the Worker route, set ACCESS_TEAM_DOMAIN + ACCESS_AUD in vars. The
  Worker verifies the Cf-Access-Jwt-Assertion header against
  https://<team>.cloudflareaccess.com/cdn-cgi/access/certs with jose — no JWT
  signing, no /auth/login//auth/callback routes needed (spec §Auth collapses to
  one verify middleware).

  skipped: routes (you're writing them), R2 fallback (add when diffs >1MB), AI
  response cache + spend cap (Stage 2).

  but it means /ai/edit output schema is yours to pin when you write that route.
